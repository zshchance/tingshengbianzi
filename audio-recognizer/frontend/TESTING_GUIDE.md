## 🧪 测试步骤

### 测试1：基础功能验证
1. 访问 http://localhost:34115
2. 观察"Vue组件已完整迁移！v2.0.0"欢迎消息
3. 点击"测试Toast通知"验证所有消息类型
4. 点击"测试进度条"验证动画效果
5. 点击"测试Wails连接"验证后端连接

### 测试2：文件处理功能
1. 拖拽一个音频文件到拖拽区域
   - 支持格式：MP3, WAV, M4A, AAC, OGG, FLAC
   - 最大大小：100MB
2. 观察文件信息显示（名称、大小、格式、时长）
3. 点击"清除文件"按钮重置

### 测试3：设置功能
1. 点击右上角"⚙️ 设置"按钮
2. 测试基础设置：
   - 切换主题（浅色/深色/自动）
   - 切换界面语言
3. 测试识别设置：
   - 更改识别语言
   - 调整置信度阈值
   - 启用/禁用词汇级时间戳
4. 点击"显示高级设置"：
   - 测试导出设置
   - 测试AI优化设置
   - 修改模型路径
5. 点击"保存设置"
6. 关闭设置窗口

### 测试4：完整识别流程
1. 选择一个音频文件
2. 点击"🎤 开始识别"按钮
3. 观察进度条和状态更新
4. 识别完成后，结果显示组件出现
5. 测试结果显示的不同标签页：
   - "📄 原始结果" - 查看识别文本
   - "✨ AI优化" - 点击后启动AI优化
   - "🎵 字幕模式" - 查看时间字幕
6. 测试导出功能：
   - 点击"💾 导出"按钮
   - 验证文件下载

### 测试5：错误处理测试
1. 尝试选择非音频文件
2. 尝试选择过大的文件
3. 在未选择文件时点击开始识别
4. 在识别过程中点击停止
